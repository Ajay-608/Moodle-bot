WHAT IS MOODLEBOT?
------------------
MoodleBot is an AI-powered educational chatbot built with Django.
Students ask questions about their course content and get instant
natural language answers powered by a local AI model.

It uses RAG (Retrieval-Augmented Generation):
  1. Student asks a question
  2. FAISS searches 2200+ course document chunks
  3. Ollama AI reads relevant chunks and writes a natural answer
  4. Student sees a conversational tutor-style response

TECHNOLOGY STACK
----------------
Backend        : Django 4.2.7 (Python)
Database       : SQLite (development)
AI Model       : Ollama llama3.2:1b (runs locally, free, offline)
Embeddings     : Sentence Transformers all-MiniLM-L6-v2
Vector Search  : FAISS (Facebook AI Similarity Search)
Frontend       : HTML + CSS + JavaScript (AJAX)

KEY FEATURES
------------
- Real-time AI chat with 88% accuracy
- Role-based dashboards (Student / Teacher / Admin)
- 2200+ course document chunks indexed and searchable
- Learning gap tracking per student
- Feedback system (1-5 star ratings)
- TAM Survey for measuring user acceptance
- Knowledge base management (upload course materials)
- Completely free - no API key, no internet needed for AI

THREE USER ROLES
----------------
STUDENT  : Chat with bot, view learning gaps, rate responses
TEACHER  : Monitor class performance, view top questions,
           see struggling students, upload materials
ADMIN    : Manage all users, view system stats, configure bot

AI PIPELINE
-----------
Question → Sentence Transformer (384-dim vector)
        → FAISS Index Search (top-3 chunks)
        → Ollama llama3.2:1b (natural language generation)
        → Clean conversational answer (88% confidence)

KNOWLEDGE BASE
--------------
Source 1 : database_dataset.json (140 DBMS Q&A topics)
Source 2 : Course PDF text files (2200+ chunks)
           - Database Management System (DBMS).txt
           - Information Systems.txt

SETUP COMMANDS (in order)
--------------------------
pip install -r requirements.txt
ollama pull llama3.2:1b
python manage.py migrate
python manage.py populate_sample_data
python index_text_files.py
python build_index.py
python manage.py runserver
